# 🚀 MERN Stack Social Media Platform

A modern, full-stack social media application built with the **MERN Stack**. Users can create and interact with posts, connect with others, exchange real-time messages, and receive notifications through a responsive and user-friendly interface.

## ✨ Features

### 👤 Authentication

* Secure user authentication powered by **Clerk**
* User profile management
* Protected routes

### 📝 Posts

* Create posts
* Edit posts
* Delete posts
* Upload up to 4 images per post
* Like posts
* Comment on posts

### 👥 Social Features

* Follow & Unfollow users
* Send connection requests
* Accept/Reject connection requests
* Discover new users
* Followers & Following lists
* Verified user badges

### 💬 Messaging

* Real-time messaging using **Server-Sent Events (SSE)**
* Instant message notifications

### 🔔 Notifications

* Background workflows powered by **Inngest**
* Connection request reminders
* Unseen message notifications

### 🎨 UI/UX

* Responsive design
* Light & Dark mode
* Clean and modern interface

# 🧠 What I Learned

Building this project strengthened my understanding of:

* Full-stack application architecture
* Authentication & Authorization
* REST API development
* MongoDB & Mongoose
* Redux Toolkit state management
* Real-time communication with Server-Sent Events (SSE)
* Image uploads and cloud storage
* Background jobs using Inngest
* Deployment and debugging production issues

One of the most challenging parts of this project was implementing **real-time messaging using Server-Sent Events (SSE)**. Managing persistent connections, handling real-time updates, solving CORS issues, and debugging deployment problems provided valuable hands-on experience.
